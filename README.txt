LearningHub - Step Up Your Learning.

LearningHub is a full-featured and interactive online education dashboard platform. It is specially designed for BCA (Bachelor of Computer Applications) students. This platform connects book knowledge with practical coding exercises and a unique reward system so that students stay motivated until the end.

(1.) Main Idea and Vision of the Project:

On normal platforms, students just watch videos and leave. But LearningHub keeps the user active using a unique Gamification and Reward Model:

Chapter-wise Study: Every course is divided into chapters in a systematic way, and every chapter has specific topics inside it.

Instant Topic Evaluation: When a topic is fully covered, there is an instant MCQ-based test which is necessary to pass.

Live Code Simulator (For Programming Courses): It has a custom console window where users get 2 coding problems for programming courses. The user writes their code, runs the test, and the system verifies the output.

Written Test and Photo Upload (For Other Courses): For non-programming courses, the user can solve answers on paper and upload a photo of it, which is checked by the backend logic.

The 30-Minute Mega Challenge: When all topics of a chapter are finished, there is a big Mega Quiz. It takes 20 to 25 minutes and has more than 30 questions.

50% Fees Refund for Top 10 Users: Inside paid courses, a dynamic student ranking is made based on the marks of the Mega Quiz and written problems. These top 10 users get half of their fees back (50% refund) along with a congratulation email.

(2.) Details of Every Page and Section of the Website:

The website is designed on a clean, single-page interface where different views open dynamically:

Login and Register Screen: It has a premium, clean login card design with an option to select Student or Admin. A user can login instantly using demo credentials.

Main Dashboard Header: At the top, there is a logo, a dynamic search bar (to search courses), and a categories dropdown (Technical, Web, Database, AI/ML). On the right side, it shows the active user's name and a logout button.

3-Line Menu Pop-up Sidebar: On the top-left side, there is a 3-line button. Clicking it opens a premium side drawer. It has options to see the Profile Section, Course Progress, Settings, Account Payment, and Admin Logs.

Course Filtering Row: Right below the search bar, there is an empty line space with three dynamic buttons: All, Free, and Paid. Clicking them automatically filters the course cards below.

Dynamic Course Cards Matrix: Every course has a separate white card with a FREE or PAID tag on the top-right side. Inside the card, you can see the course name, short description, category, and price. Clicking the card takes the user straight inside the course module.

(3.) Technology Stack Used:

HTML5: The structure of the website is made using semantic tags like Header, Aside, Main, and Section.

CSS3: It uses a custom Indigo Blue color theme, clean card shadows, flexible layout, and is fully mobile-responsive.

JavaScript (ES6): It handles the login validation, dynamic switching of tabs, real-time search functionality, and category filters without reloading the page.
